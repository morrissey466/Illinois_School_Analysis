Illinois School Analysis

Project Overview

This project will use data provided by Illinois State Education Board to determine which key features from Report Card Data Library strongly correlate with student performance. The regression will quantify a relationship of student performance with both ELA and Math scores. The key factors we chose to go with were proficiency from low-income families, total students (with breakdown by race and age), attendance rate, mobility rate, dropout and graduation rates, etc. This project will look at 487 districts from the state of Illinois. They have small, medium and large. There are two main features that will be used to analyze this project districts and high schools. Below are tableau visualizations from the data that was gathered.
