Illinois School Analysis

Project Overview

This project used data provided by Illinois State Education Board to determine which key features from Report Card Data Library strongly correlate with student performance. The regression will quantify a relationship of student performance with both ELA and Math scores. The key factors we chose to use were proficiency from low-income families, total students (with breakdown by race and age), attendance rate, mobility rate, teacher retention rate, dropout and graduation rate. This project will look at 487 districts from the state of Illinois. The districts are split into three district sizes small, medium and large. There are three main factors that are used to analyze this project districts, financials and high schools. Below is a Tableau dashboard that will assist in visualizing the locations and sizes of the districts, class size, teacher expenditure rate, chronic absenteeism, low income, teach salary, and college remediation.
